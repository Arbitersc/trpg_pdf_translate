# TRPG PDF 翻译器

针对桌上角色扮演游戏（Tabletop Role Playing Game，缩写： TRPG）的PDF、text等文本书籍，通过大语言模型翻译生成结果文件。

## 项目概述

TRPG PDF Translator 是一个专为桌面角色扮演游戏(TRPG)PDF文档设计的翻译工具。它提供了完整的翻译工作流，包括PDF解析、智能翻译、术语表管理和双语对齐等功能。

## 功能特性

### 📄 PDF解析
- 支持本地PDF文件和URL
- 多种输出格式：纯文本、Markdown、JSON
- 可配置的解析选项（滑动窗口、移除图片链接等）

### 🌐 智能翻译工作流
- 自动提取文档中的专有名词
- 自动汇总名词生成译名表
- 使用大模型进行文本翻译
- 根据译名表更新译文结果
- 导出译文结果（支持单语、双语对照格式）

### 📚 术语表管理
- 从PDF提取专有名词
- 生成和管理翻译术语表
- 支持术语表导入导出

### 🔄 双语对齐
- 对齐英文和中文文本
- 生成对比视图
- 支持多种输出格式

### ⚙️ 配置管理
- 设置API密钥和模型参数
- 配置解析器选项
- 管理应用程序设置
- 支持配置导入导出功能

```bash

```

```bash

python -m src.frontend.cli.main --interactive
```

## 技术栈

### 后端架构
- **语言**: Python
- **框架**: Flask
- **PDF解析**: 集成PDF解析功能，支持文档文本提取
- **LLM集成**: 通用大模型API配置，支持OpenAI、Ollama等主流大模型平台协议

### 前端架构（规划中）
- **框架**: Vue + React + Vite + PDF.js
- **架构**: 前后端分离

### CLI工具
- **入口**: `src/frontend/cli/main.py`
- **交互界面**: `src/frontend/cli/interactive.py`
- **工作流**: `src/frontend/cli/workflow.py`
- **配置管理**: `src/frontend/cli/config.py`

## 配置要求

### API密钥配置

在使用翻译功能前，需要配置API密钥：

1. 在主菜单中选择选项5（配置管理）
2. 选择"设置API密钥"
3. 输入您的API密钥

### 支持的API提供商

- SiliconFlow (默认)
- OpenAI
- 其他兼容OpenAI API的提供商

### 配置目录结构

```
~/.trpg_pdf_translator/
├── config.json          # 应用程序配置（JSON 格式）
└── .env                 # 环境变量（包含 API 密钥等敏感信息）
```

## 故障排除

### 常见问题

**Q: 启动时出现导入错误**
A: 确保在项目根目录下运行，或正确设置Python路径

**Q: API密钥设置失败**
A: 检查API密钥格式是否正确，网络连接是否正常

**Q: PDF解析失败**
A: 检查PDF文件路径是否正确，文件是否可读

### 日志和调试

如需调试信息，可以设置环境变量：

```bash
export TRPG_PDF_DEBUG=1
./trpg-pdf
```
